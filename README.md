# hotelReservationsProject
A school project using python to create a basic hotel reservation system.

## Overview
It takes a start and end date and then shows the rooms available at the time that also match the room type selected.

## Hotel.py
It consists of a tkinter window that has a button which opens a Toplevel window with Dateentrys that allow for the date selection as well as a optionsmenu that allows for selection of the room type. When an option in the Optionsmenu isn't selected rooms with either type will be shown, but if dates aren't selected reservation will not be allowed.
